# AI-ML-Internship-Task2

## Objective
Predict future stock prices (short-term) using regression models.

## Dataset
Historical stock market data fetched via [Yahoo Finance](https://finance.yahoo.com) using the `yfinance` Python library.

## Models Applied
- Linear Regression
- (Optional) Random Forest Regressor

## Key Results
- Plotted actual vs predicted closing prices
- Evaluation metrics:
  - Mean Absolute Error (MAE):0.7572127520659364
  - Root Mean Squared Error (RMSE): 0.9658203163445872

## Insights
- The model captures the general trend of stock prices.
- Predictions are close to actual values, but some fluctuations are missed.
- Random Forest may improve accuracy compared to Linear Regression.
